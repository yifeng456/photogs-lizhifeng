# 照片存放说明

## 目录结构

```
public/photos/
├── landscape/          ← 风光类原图
├── street/             ← 街拍类原图
├── portrait/           ← 人像类原图
├── night/              ← 夜景类原图
└── thumbnails/         ← 缩略图（所有分类统一放这里）
```

## 上传步骤

### 1. 放入照片
将你的照片原图按分类放入对应目录，如 `public/photos/landscape/sunset.jpg`

### 2. 生成缩略图
```bash
node scripts/generate-thumbnails.js
```
此脚本会自动为 `public/photos/` 下所有原图生成缩略图，输出到 `thumbnails/` 目录。
需要先安装依赖：`npm install sharp`

### 3. 修改数据文件
编辑 `src/data/photos.js`，将占位数据改为你的真实照片信息：
```js
{
  id: 1,
  title: '你的作品标题',
  description: '作品描述',
  src: '/photos/landscape/sunset.jpg',           // 原图路径
  thumbnail: '/photos/thumbnails/sunset.webp',   // 缩略图路径
  // ... 剩余字段
}
```

### 注意事项
- 建议原图宽度 ≤ 4000px，保证网页加载速度
- 单张原图大小不要超过 50MB（GitHub 限制 100MB）
- 缩略图建议宽度 600px，WebP 格式
- 确保文件名不含中文和特殊字符（建议：sunset-mountain.jpg）
